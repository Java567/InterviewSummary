# 快速排序

## 一、思想

- 首先选定一个基准元素pivot，可以随机将数组的值赋给它，也可以将数组第一个元素值赋给它
- 设置一个mark值，它一开始指向数列起始位置，这个mark指针代表小于基准元素pivot中的值
- 然后进行遍历，在遍历过程中，如果遍历到的元素大于基准元素，则继续往后遍历
- 如果遍历到的元素小于基准元素，那就要做两个事，第一，将mark右移（就是将mark++）,第二，将遍历到数组中位置的元素值和mark所在的值（arr[mark]）进行交换，然后继续往后遍历
- 遍历完之后，将基准元素值与arr[mark]进行交换，并将mark返回
- 之后再利用递归的方法，将其排序
